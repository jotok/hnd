# hnd

An implementation of the hawk and dove game as an agent-based model. See [http://tokle.us/programming/2014/07/05/the-hawk-and-dove-game/](http://tokle.us/programming/2014/07/05/the-hawk-and-dove-game/).

## Usage

    lein run -m hnd.core
